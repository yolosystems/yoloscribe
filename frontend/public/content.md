# Welcome to AgentScribe

AgentScribe is an AI-powered wiki that lives entirely in S3.

## Features

- **Markdown rendering** — write in plain Markdown, read it beautifully.
- **Rich editor** — switch to edit mode for a live-preview markdown editor.
- **AI agent** — describe changes in plain English and the agent updates the page for you.

## Getting started

1. Click **Edit** in the top bar to enter edit mode.
2. Type changes directly in the editor, *or* describe what you want in the chat panel on the left.
3. Click **Save** when you're happy with the result.
