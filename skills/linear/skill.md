# Skill: linear

You have access to Linear project management tools provided by the Linear MCP server.

## When to use this skill

Use these tools when the user wants to:
- View, create, or update Linear issues
- Manage projects, cycles, and teams in Linear
- Triage or prioritise a backlog
- Summarise the state of a project or sprint
- Create issues from wiki content or meeting notes

## Authentication

This skill uses OAuth. Users must authenticate via the Credentials panel before
agents using this skill can run.

## Usage guidelines

- When creating issues, always confirm the team and project with the user first
  unless they have already specified them.
- Use `list` tools to discover available teams, projects, and states before
  creating or updating records.
- Prefer updating existing issues over creating duplicates — search before creating.
