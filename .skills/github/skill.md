Read and write GitHub resources: issues, pull requests, file contents, commits, and code search.

Use this skill when an agent needs to:
- Look up or create GitHub issues and pull requests
- Read source files or directory listings from a repository
- Search code across repositories
- Summarise recent activity (commits, PRs merged, issues opened)

Requires a `GITHUB_TOKEN` secret (classic PAT or fine-grained token with repo scope).
