# Tool: slack

You have access to Slack tools provided by the official Slack MCP server.

## Use cases

Use this tool when the user wants to:
- Send messages to Slack channels or direct messages
- Read recent messages from a channel
- Search message history across the workspace
- Look up users, channels, or workspace members
- React to messages or manage threads

## Authentication

This tool uses OAuth. Users must authenticate via the Tools panel before
agents using this tool can run.
